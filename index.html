Welcome to SJdownloader.

SJdownloader is a very light weight desktop downloader for downloading content from internet. It is simple, easy to use and yet fast. Just give it a try and let me know .

Current Features

  • Light weight and easy to use.
  • Speedy downloads (using segments for download)
  • Set your preferences like default location, number of segments per download, history management.
  • Default regex to sort you downloads for popular formats like m4, mp3, png, jpeg, jpg etc
  • Custom regex option for advance filtering.
